Лабораторна робота 1 з JavaScript

Застосування JavaScript для розв’язання прикладних задач за допомогою консолі браузера

Роботу виконала студентка групи КН-32 Синишин Вікторія

У цій лабораторній роботі реалізовано функцію triangle(), яка розв’язує прямокутний трикутник. Функція обчислює всі сторони (a, b, c) та гострі кути (alpha, beta) за двома заданими елементами.

Підтримуються такі типи даних:

  • leg — катет
  • hypotenuse — гіпотенуза
  • adjacent angle — кут, прилеглий до заданого катета
  • opposite angle — кут, протилежний до заданого катета
  • angle — гострий кут при заданій гіпотенузі
